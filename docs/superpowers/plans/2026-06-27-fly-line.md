> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Architecture:** Implement `FlyLineEffect` in the existing core SDK file using entity polylines, sampled high-altitude quadratic arcs, and callback-driven animated trails. Wire the demo and knowledge layer exactly like existing effects.

**Tech Stack:** TypeScript, Cesium entity API, Node test runner, Vite demo.
